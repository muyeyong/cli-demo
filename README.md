# lvluo 脚手架
## 简介

基于 Vue Cli 生成，基本架构是 Vue3.0 + TS + Eslint(airbnb)，保存代码会自动格式化

UI组件使用的antd，支持按需引用

## 特色
支持微前端(qiankun)

自动导入视图路由

自动导入store

提交代码自动检查代码是否规范

## 使用

```js
npm install lvluo -g
```

使用帮助：`lvluo -h`

创建组件：` lvluo addCmp componentName`

创建视图: `lvluo addView viewName`

创建store：`lvluo createStore storeName`
